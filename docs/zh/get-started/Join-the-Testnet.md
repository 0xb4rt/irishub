# IRISnet测试网

## Fuxi测试网

Fuxi测试网将用于测试irishub的最新功能。


## 如何加入fuxi测试网

### 第一步: 安装IRIScli

请根据以下[教程](Install-the-Software.md) 在服务器上完成**Iriscli**的安装。

### 第二步: 接入测试网节点

* RPC: https://rpc.testnet.irisnet.org

* LCD: https://lcd.testnet.irisnet.org/swagger-ui/
