# iriscli gov deposit

## 描述
 
抵押保证金以激活提议
 
## 使用方式
 
```
iriscli gov deposit <flags>
```

打印帮助信息:

```
iriscli gov deposit --help
```


## 特殊标志
 
| 名称, 速记     | 默认值 | 描述             | 是否必须  |
| ------------- | ----- | --------------- | -------- |
| --deposit     |       | 发起提议的保证金   | 是       |
| --proposal-id |       | 抵押保证金的提议ID | 是       |

## 例子

### 抵押保证金

```shell
iriscli gov deposit --chain-id=<chain-id> --proposal-id=<proposal-id> --deposit=50iris --from=<key_name> --fee=0.3iris
```
