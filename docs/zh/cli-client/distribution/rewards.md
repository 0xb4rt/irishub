# iriscli distribution rewards

## 介绍

查询验证人或委托人的所有收益

## 用法

```
iriscli distribution rewards <address> [flags]
```

打印帮助信息：
```
iriscli distribution rewards --help
```

## 示例

```
iriscli distribution rewards <address>
```
执行结果示例：
```json
[
  {
    "denom": "iris-atto",
    "amount": "8652437451742091250"
  }
]
```
